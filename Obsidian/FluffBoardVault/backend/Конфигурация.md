# Конфигурация

Parent: [[Index]]

## Назначение

`BoardOptions` описывает единственную репозиторию, путь к SQLite и локальных пользователей. `DotEnv` до запуска ASP.NET Core загружает переменные из `.env`.

Файл всегда берётся из корня FluffBoard. При обычном запуске он дополняет только отсутствующие переменные окружения. В режиме `--mac` значения корневого `.env` имеют приоритет, а отсутствующие в нём параметры остаются из окружения процесса (например, контейнера).

При запуске backend печатает путь к загружаемому `.env`, распознанные ключи и итоговые `Board:Repository:Owner` / `Name` до валидации. Значения ключей с `Token` или `Password` не выводятся.

```bash
dotnet run --project backend -- --mac
```

## Настройки

| Переменная | Назначение |
|---|---|
| `GitHub__Token` | Token отдельного GitHub service account. |
| `Board__Repository__Owner` | Owner единственной репозитории доски. |
| `Board__Repository__Name` | Имя единственной репозитории доски. |
| `Board__DatabasePath` | Необязательный путь к SQLite; по умолчанию `data/fluffboard.db`. |
| `Board__Users__N__Username` / `Password` | Локальные учётные данные пользователя доски. |
| `Board__Users__N__GitHubLogin` / `TelegramId` | Связанные идентичности этого пользователя. |

Пример всех значений находится в [`.env.example`](../../../.env.example); настоящий `.env` игнорируется Git.
