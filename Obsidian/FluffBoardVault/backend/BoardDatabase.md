# BoardDatabase

Parent: [[Index]]

## Назначение

`BoardDatabase` хранит в локальной SQLite-базе пользователей, которым разрешён вход на доску. В записи пользователя объединены локальный логин, GitHub login и будущий Telegram ID.

## Ключевые методы

| Метод | Описание |
|---|---|
| `InitializeAsync(...)` | Создаёт таблицу и точно синхронизирует пользователей из конфигурации, отзывая доступ у удалённых из `.env`. |
| `AuthenticateAsync(...)` | Проверяет пароль пользователя и возвращает его локальный профиль. |
| `GetUserAsync(...)` | Загружает профиль для восстановления cookie-сессии. |
| `GetProfilesAsync(...)` | Возвращает безопасный список пользователей для выбора исполнителя. |
| `PasswordHasher` | Создаёт и сравнивает PBKDF2-хеши с random salt и constant-time сравнением. |

Пароль не возвращается в API и не хранится в SQLite открытым текстом.
