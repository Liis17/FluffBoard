# Приложение

Parent: [[Index]]

## Назначение

React-клиент FluffBoard запрашивает issues выбранного GitHub-репозитория через backend и отображает их заголовки, labels и assignees. При ошибке показывает сообщение API.

## Файлы

- `frontend/src/main.jsx` — точка входа, монтирует `App` в `#root` внутри `StrictMode`.
- `frontend/src/App.jsx` — компонент экрана и загрузка ответа API.
- `frontend/src/App.css` — стили контейнера и элементов экрана.
- `frontend/src/index.css` — глобальные CSS-правила и базовая тема.
- `frontend/vite.config.js` — подключает React-плагин Vite.
- `frontend/package.json` — зависимости и команды разработки, сборки, линтинга и preview.
- `frontend/index.html` — HTML-документ Vite.
- `frontend/public/` и `frontend/src/assets/` — статические графические ресурсы.

## Ключевые методы/функции

| Метод | Описание |
|-------|----------|
| `App()` | Хранит owner, repository, состояния загрузки и список issues; рендерит форму и результаты. |
| `loadIssues(event)` | По submit запрашивает endpoint GitHub backend и обрабатывает успех или ошибку. |
| `createRoot(...).render(...)` | Монтирует React-дерево в DOM. |

## Зависимости

- Использует: React, React DOM, Vite и [[backend/API]].
- Используется в: браузере через Vite development server или собранные assets.

## Важные детали

URL API задан напрямую как `http://localhost:5279/api/github/repos/{owner}/{repository}/issues`. После изменения адреса backend его нужно синхронно обновить здесь и в [[backend/API]].
