# FluffBoard — База знаний

FluffBoard — минимальное full-stack приложение: React 19 + Vite на клиенте и ASP.NET Core на .NET 10 на сервере. Клиент показывает ответ backend API.

## Правила обновления базы знаний

При работе с проектом обновляй соответствующую заметку, если изменилась архитектура, добавлен компонент, изменились зависимости или конфигурация. Новый компонент → `{Domain}/{Name}.md` + ссылка здесь. Для wikilinks используй двойные квадратные скобки вокруг имени заметки или пути к ней.
